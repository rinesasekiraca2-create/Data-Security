Breaking Caesar Cipher using Frequency Analysis

📌 Project Overview

This project demonstrates how to break the Caesar Cipher using letter frequency analysis.
The goal is to decrypt an encrypted message by comparing the frequency of letters in the ciphertext with known letter frequencies from a reference text file.
The implementation is done in Python and highlights basic concepts of cryptography and cryptanalysis.

🎯 Objectives

Implement Caesar Cipher encryption and decryption
Analyze letter frequency from a given text file
Use frequency analysis to guess the encryption key (shift)
Automatically decrypt ciphertext without knowing the key
